# Producto

InmoSignal ayuda a inmobiliarias a definir busquedas de captacion y recibir oportunidades privadas a partir de listings capturados globalmente.

## Principios

- No existe scraper por cliente.
- El scraper no conoce tenants concretos.
- Las busquedas del cliente se traducen a demandas tecnicas.
- Las demandas se agrupan en planes globales de scraping.
- Los listings se guardan de forma global.
- El matching decide que tenant ve cada listing.
- El CRM nace alrededor de oportunidades, no solo inmuebles.
