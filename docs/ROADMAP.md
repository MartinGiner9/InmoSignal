# Roadmap

## Fase 0 - Base tecnica

- Monorepo, apps y packages.
- Prisma schema inicial.
- Docker Compose con PostgreSQL y Redis.
- Adapters de scraping stub.

## Fase 1 - Producto minimo

- CRUD de tenants, usuarios y busquedas.
- Generacion de demandas tecnicas.
- Planificador inicial de scraping con mock adapter.
- Matching simple y oportunidades.

## Fase 2 - Operacion

- Scraping real revisado fuente por fuente.
- Observabilidad de runs y errores.
- Roles, limites por plan y audit logs.

## Fase 3 - CRM ampliado

- WhatsApp, llamadas y scoring.
- IA para priorizacion y resumen.
- Automatizaciones comerciales.
